# Task

Write a bash script named `topwords.sh` in this directory that reads
`article.txt`, lowercases it, strips punctuation, splits on whitespace, and
prints the 3 most frequent words with their counts, one per line, in the
format `word count`, most frequent first (ties broken alphabetically).
Ignore the stopwords: "the", "a", "an", "and", "of", "to", "in", "is", "it".

Only create `topwords.sh`. Do not run it yourself; the check script runs it.
